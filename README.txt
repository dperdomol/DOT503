For the automation of the creation of the complete package of the program, the Gulp library was used, this is a library that allows us to 
automate processes. In order to run correctly, it is necessary to install the following programs on the system:

1. Node.js
2. NPM

The following line must be executed after the installation of the programs described above:
IMPORTANT: BEFORE RUNNING THIS COMMAND, THE CONSOLE MUST BE POINTED AT THE ROOT FOLDER OF THE PROJECT

1. npm install

This will locally install all the dependencies needed to run gulp.
After doing all the previous execution you can run the following command:

1. gulp

This command will automatically create the package ready for production.
